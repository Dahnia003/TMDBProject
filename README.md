**Overview**

This project collects and analyzes data from The Movie Database (TMDB) using Python and the TMDB API.
It builds a historical dataset of popular movies and TV shows from the past few months and prepares the data for use in a Power BI dashboard.

*The goals are simple:*

-Automatically fetch and refresh data weekly  
-Transform and clean the data into CSV files  
-Create genre level, popularity, and language insights  
-Load everything into Power BI for interactive visual analysis
